# Houston Pest Control

A professional web application for a local pest control service in Houston. Built with React, Vite, and Tailwind CSS.

## Tech Stack
- React
- Vite
- Tailwind CSS

## Running Locally

To run the application locally:
```bash
npm install
npm run dev
```

## Deployment

This website is configured to be deployed to GitHub Pages via the `gh-pages` branch.
